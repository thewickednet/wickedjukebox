MpdController

MpdConnection([host="localhost"[, port=6600[, timeout=20]]])

This is basically the C mpd_Connection struct, merged with all of the C functions that acted upon it, in a single class, with of course some changes to make it more Pythonic. However, I tried to keep it fairly close to the C implementation. Additional convenience stuff, and much more, is in MpdController.
