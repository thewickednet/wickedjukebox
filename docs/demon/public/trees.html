Module Hierarchy

    • model

    • players
      • icecast
      • mpd: An interface to the music player daemon (mpd).
      • skeleton: A no-op player interface.
    • playmodes
      • queue_positioned: Strict FIFO Queuing (ordered by position) Pops off the song with position 1, then substracts 1 from the position field of each item and finally removes all items with an id smaller than -10
      • queue_skeleton: This is a no-op playmode used as blueprint for new playmodes.
      • queue_strict: Strict FIFO Queuing.
      • random_skeleton: This is a no-op playmode used as blueprint for new playmodes.
      • random_weighed: Weighed random.
      • random_weighed_prefetch: Weighed random.
    • plparser: Methods to convert a simplified filter string into an SQL query
    • pltest: Methods to convert a simplified filter string into an SQL query
