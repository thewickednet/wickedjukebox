Module demon.playmodes.queue_skeleton

This is a no-op playmode used as blueprint for new playmodes. Simply complete the methods in this module and make them return the right type and you should be fine.

Function Details

dequeue()

Return the next song from the queue
Returns:
This should return a valid "Song" instance, which points to the next song on the queue. If nothing was on the queue, or no song could be determnined, then return "None", so the system can fall back to random mode.
           (type=Song)
