Type Song

object --+

InfoEntity --+

            Song

An InfoEntity subclass representing a music file somewhere inside of mpd's music directory. Each Song instance will have a path attribute, but other attributes may remain as empty strings, since not all files will have the corresponding metadata (id3 tags, etc) needed to obtain the information.
