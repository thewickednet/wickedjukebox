Player Modules¶

This module contains implementations for the underlying player backends

class wickedjukebox.component.player.MpdPlayer(config: Optional[wickedjukebox.config.Config], channel_name: str)

A bridge to a MPD daemon.

Warning

If the MPD daemon runs on a different host than the jukebox, and if the root file-path to the MP3 folder differs, the config-option path_map has to be used to translate these paths.

Configuration Example¶
[channel:example:player]
type = mpd
host = 127.0.0.1
port = 6600
path_map = /path/to/local/files:/path/to/mpd/files
class wickedjukebox.component.player.NullPlayer(config: Optional[wickedjukebox.config.Config], channel_name: str)

A player which only logs the underlying calls but does nothing.
