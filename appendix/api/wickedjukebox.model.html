wickedjukebox.model.audiometa module#

Abstraction Layer for mutagen

Mutagen exposed the internal metadata as stored in the original file. This causes an inconsistent external interface (f. ex. the title field is not accessed in the same way for MP3 files as for FLAC files).

This module provides the necessary abstraction to handle all files identically

This modul is also executable and can be used to display meta information of a specific file.
